📚 QPredictor – AI-Based Exam Question Paper Prediction System

QPredictor is an AI-powered web application that analyzes previous years' university question papers to identify recurring patterns and predict the most probable questions for upcoming examinations.

The platform helps students prepare smarter by providing AI-driven insights instead of manually analyzing hundreds of previous papers.

---

🚀 Features

🔐 User Authentication
- Secure Signup & Login
- JWT Authentication
- Password Hashing

📄 Question Paper Management
- Upload PDF question papers
- Automatic text extraction
- OCR support for scanned PDFs (Work in Progress)
- Organize papers subject-wise

🤖 AI Prediction Engine
- Analyze multiple previous papers
- Detect repeated questions
- Frequency-based prediction
- Important topic identification

📊 Analytics Dashboard
- Paper statistics
- Prediction confidence
- Question frequency analysis

💬 AI Chat Assistant
- Ask subject-related questions
- Get AI-powered explanations
- Interactive learning experience
